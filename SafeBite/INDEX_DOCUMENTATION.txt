✨ POINTS CLÉS
─────────────

POUR COMPRENDRE L'INTÉGRATION:
  • Restaurant s'inscrit → enregistré dans AdminService
  • Client s'inscrit → enregistré dans AdminService
  • Admin voit les demandes et les clients
  • Admin peut valider/refuser restaurants
  • Admin peut suspendre/supprimer clients

POUR UTILISER LE SYSTÈME:
  1. Compiler: javac -d out -sourcepath src src/main/Main.java
  2. Lancer: java -cp out main.Main
  3. Suivre GUIDE_TEST.txt pour tester

POUR MODIFIER LE SYSTÈME:
  1. Lire MODIFICATIONS_DETAIL.txt
  2. Lire INTEGRATION_GUIDE.md
  3. Respecter l'ordre d'initialisation (AdminService avant AuthService)
  4. Tester après modification
