## 🧠 Project Overview

This project is a **full-stack web/mobile application** designed to help people suffering from **Celiac Disease**.

The platform allows users to:
- Discover gluten-free dishes 🍽️
- Search certified restaurants 📍
- Place safe food orders 🛒
- Leave reviews and ratings ⭐
- Receive personalized recommendations 🎯

---

## 💡 Objective

To build a smart ecosystem connecting:
- Clients (gluten-free users)
- Restaurants (offering safe meals)
- Administrators (content moderation & validation)

---

## ⚙️ Tech Stack

- Frontend: React / HTML / CSS
- Backend: Node.js / Spring Boot / PHP
- Database: MySQL / PostgreSQL
- Auth: JWT
- Tools: GitHub, Postman

---

## 🚀 Git Branch Strategy

Each developer works on an independent feature branch:

- `feature/authentification` → Login / Signup system  
- `feature/visiteur` → Public browsing (plats & restaurants)  
- `feature/restaurant-management` → Menu & orders management  
- `feature/client-space` → Orders, reviews, recommendations  
- `feature/admin-dashboard` → Admin panel (users, validation, moderation)

---

## 🎯 Final Goal

Deliver a complete platform that improves the daily life of people with celiac disease by ensuring safe, reliable, and enjoyable food experiences.
